Pourquoi ça marche enfin

Ton éditeur de signature (Tiptap) supprime les <div> et les fonds de cellule, mais conserve les images. Le trait doré est donc une petite image 3×104 px (même principe que les icônes réseaux qui, elles, s’affichaient déjà). Colle ce code via le bouton « Coller votre signature (HTML) ».
